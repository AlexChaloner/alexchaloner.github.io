Learning the learning rate

What happens when every feature is allowed to decide how quickly it should learn?

Stochastic gradient descent gives every feature the same learning rate. Incremental Delta-Bar-Delta (IDBD) starts there too, then adapts a separate rate for every feature using the history of its updates.

Reading the experiment. The green bar is the only coefficient that should approach 1. Every other coefficient should stay near zero. “Clean RMS” evaluates against the predictable target with the target noise removed.

At batch size 1, this is classic online IDBD. Larger batches use a mean-gradient diagonal approximation and are best read as an exploration, not the original 1992 rule.
