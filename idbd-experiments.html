Stochastic gradient descent gives every feature the same learning rate. Incremental Delta-Bar-Delta (IDBD) starts there too, then adapts a separate rate for every feature using the history of its updates.

Ready

Computed live in this browser. Following the Oak Lab stream, every feature activates independently with 1% probability; feature 1 is the only predictive feature. The browser-sized version uses 64 rather than 4,096 features. Training yields between animation frames, and changing a setting cancels the current run.

Reading the experiment. The learning-rate and model panels use the same feature order, with the predictive feature first in green. In the model panel, that green coefficient should approach the reference line at 1 while every other coefficient stays near zero. Prediction MSE is measured before each update against the noisy target; the curve is an exponential moving average over minibatches. Axes stay fixed between runs; red edge labels flag values outside their range.

At batch size 1, this is classic online IDBD. Larger batches use a mean-gradient diagonal approximation and are best read as an exploration, not the original 1992 rule.
